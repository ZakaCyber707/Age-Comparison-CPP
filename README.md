# Age Comparison Tool
A simple C++ console application that compares a user's age against a hardcoded value (20) and determines if they are older, younger, or the same age.

## How to Run

### Option 1: Using an IDE (Visual Studio / Dev-C++)
1. Open your IDE (e.g., **Visual Studio** or **Dev-C++**).
2. Create a new C++ Project.
3. Copy the code from `AgeComparison.cpp` into your project's main file.
4. Press **F5** (Visual Studio) or **F11** (Dev-C++) to Compile and Run.

### Option 2: Using Command Line (MinGW/GCC)
1. Open your terminal or command prompt.
2. Navigate to the folder containing the file.
3. Compile using: `g++ AgeComparison.cpp -o AgeCompare`
4. Run the executable: `AgeCompare.exe`
